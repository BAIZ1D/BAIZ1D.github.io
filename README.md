# Baizid Al Hamid - Portfolio & CV Site

Personal portfolio website and dynamic CV generator built for Baizid Al Hamid. High-contrast Neo-Brutalist design that showcases professional experience, projects, and publications, with integrated print stylesheet overrides to generate standard 1-page resume PDFs directly from the web layout.

Live site: https://baiz1d.github.io/

## Features

- **Neo-Brutalist Layout:** High-contrast color scheme, thick solid borders, drop shadows, and hardware-accelerated CSS spring animations.
- **Infinite Marquee Ticker:** Top viewport marquee banner rendering navigation/branding headlines.
- **Micro-Interactions:** CSS transition hover states on experience and project cards.
- **Automated CV Export:** Responsive CSS print media queries mapping portfolio content into a standard single-page professional resume layout via the "Download CV" action.
- **Dynamic Chronology:** Real-time graduation logic (handling master degree timeline conversions) and automated copyright updates.
- **Lightweight Architecture:** Vite-based build process with React 19, TypeScript, and Tailwind CSS.

## Tech Stack

- **Framework:** React 19 (TypeScript)
- **Styling:** Tailwind CSS (Vanilla CSS transition transforms)
- **Icons:** Lucide React
- **Build Tool:** Vite

## Getting Started

### Prerequisites

Node.js (LTS version recommended).

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/BAIZ1D/BAIZ1D.github.io.git
   cd BAIZ1D.github.io
   ```

2. Install the dependencies:
   ```bash
   npm install
   ```

3. Run the development server locally:
   ```bash
   npm run dev
   ```

4. Build production assets:
   ```bash
   npm run build
   ```
